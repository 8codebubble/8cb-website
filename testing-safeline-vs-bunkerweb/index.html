WAF Testing: Safeline or BunkerWeb

ymcadmin December 12, 2024 Containers
A Web Application Firewall (WAF) is a security solution designed to protect web applications by filtering and monitoring HTTP traffic between a web application and the internet. It acts as a shield, safeguarding applications from common vulnerabilities and attacks such as:

SQL Injection: Prevents unauthorized access to databases.

Cross-Site Scripting (XSS): Blocks malicious scripts from being executed in a user's browser.

File Inclusion Attacks: Protects against unauthorized access to files on a server.

I'm currently testing Safeline and BunkerWeb to see which best suits my current home lab services and security infostructure. Both feature reverse proxy options but I much prefer to keep my pfsense(Ha proxy) implementation which give me greater flexibility for internal and external traffic.
